## Write Basic Git Commands Below

- Clone a Repository:
```
$ git clone *pasted url*
```

- Check Status:
```
$ git status
```

- Stage (Add):
```
$ git add -A
```

- Commit:
```
$ git commit -m "*message*"
```

## Write Basic GitHub Commands Below

- Push:
```
$ git push origin master
```

- Check Remote:
```
$ git remote *usually to identical links (fetch and push)*
```
